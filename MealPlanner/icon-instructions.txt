ICON SETUP INSTRUCTIONS
========================

For the app to work as a PWA on iPhone, you need icon files.

OPTION 1 - Quick & Easy (Use an emoji):
Go to this website: https://favicon.io/emoji-favicons/fork-and-knife/
- Download the favicon package
- Extract and rename:
  - android-chrome-192x192.png ‚Üí icon-192.png
  - android-chrome-512x512.png ‚Üí icon-512.png
- Place both files in this folder

OPTION 2 - Custom Icon:
1. Create or find a square image (at least 512x512px)
2. Use https://realfavicongenerator.net/ to generate all sizes
3. Get the 192x192 and 512x512 versions
4. Rename them to icon-192.png and icon-512.png
5. Place in this folder
